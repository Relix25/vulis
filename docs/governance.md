# Governance

The canonical governance document lives at the repository root:
[../GOVERNANCE.md](../GOVERNANCE.md).

Summary of roles and decision process:

| Role | How obtained |
|---|---|
| **Contributor** | Anyone with a merged PR. |
| **Maintainer** | Sustained, high-quality contributions; invited by current maintainers. |
| **Lead maintainer** | Currently the founder (Basti); succession by maintainer vote. |

Decisions use **lazy consensus**, with ADRs for non-trivial or architectural
changes. The lead maintainer breaks ties.
