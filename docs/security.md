# Security

The canonical security policy lives at the repository root:
[../SECURITY.md](../SECURITY.md).

**To report a vulnerability, do not open a public issue.** Use GitHub's
security advisory flow instead:

> <https://github.com/vulis/vulis/security/advisories/new>

Acknowledgement within 5 business days. Only the latest `main` and the most
recent release tag are supported.
